## üìÇ Repository Structure

This repository is organized into several focused learning modules:

### [1. Data Structures & Algorithms Practice](./dsa-practice)
- **Goal**: Master fundamental Computer Science concepts.
- **Contents**: Arrays, Sets, Maps, Stacks, Queues, Linked Lists, and problem-solving patterns.

### [2. Next.js Learning](./next-js-learning)
- **Goal**: Deep dive into the React framework for production.
- **Contents**: Projects like a Blog Client using modern Next.js features.

### [3. Node.js & Express Resource](./node-express-rcs)
- **Goal**: Understand the backend runtime and framework.
- **Contents**: Core modules (fs, path, os), raw Node.js servers, and Express.js REST APIs.

### [4. Prisma ORM Exploration](./prisma-orm-exploration)
- **Goal**: Learn next-generation Node.js and TypeScript ORM.
- **Contents**: Database modeling, queries, and integration with a Blog App.

### [5. RDBMS & SQL (PostgreSQL)](./rdbms-sql-postgres)
- **Goal**: Master relational database concepts and raw SQL.
- **Contents**: Comprehensive SQL scripts covering Joins, Triggers, Indexing, Functions, and more.

### [6. TypeScript Playground](./typescript-playground)
- **Goal**: Explore static typing for JavaScript.
- **Contents**: Theory and practice with Types, Interfaces, Generics, and advanced patterns.
