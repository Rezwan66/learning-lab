# Learning Prisma ORM

Migrate all prisma schemas to SQL database (Migrate our created schema to local PostgreSQL database):

```powershell
    npx prisma migrate dev
```

Generate all possible types for our tables:

```powershell
    npx prisma generate
```

Run Prisma Studio:

```powershell
    npx prisma studio
```

Run a TS file on command line prompt:

```powershell
    npx tsx crud.ts
```
