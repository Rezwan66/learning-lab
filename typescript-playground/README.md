## Typescript Types

Primitive data types in TS include:

- number
- string
- boolean
- null
- undefined

Non-primitive data types in TS include:

- object
- array
- tuple
- enum
- never
- unknown

Other types:

- Literal
- Optional
- Function
- Union
- Intersection
- Alias
- Generic
- Rest & Spread
- Ternary
- Nullish Coalescing
- Optional Chaining
- Nullable
